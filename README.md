# 欢迎阅读Cora的CPP游戏项目

## 项目介绍

这是一个基于C++的控制台文字游戏，是Cora用来练习C++编程的项目  
当然你也可以用来学习参考，提升你阅读代码的水平和了解C++的一些内容  
如果你有什么好点子，欢迎将你的游戏内容坐进本项目并推送给我，我将会尽力将它加入到项目中  

## 代码与注释规范

1. 代码文件名全部使用小写字母，多个单词使用下划线连接
2. 类名使用驼峰命名法，变量名使用小写字母，多个单词使用下划线连接
3. 代码缩进使用4个空格，不要使用制表符
4. 禁止使用goto语句

## 提交注意

1. 提交代码时请注意遵守代码规范
2. 提交代码时请附带有详细的注释，并在结尾处换行，以--changed by somebody 结尾
3. 请不要提交无用的代码，如无必要，请不要提交编译错误的代码
4. 如果您需要提交您的代码，请先上传到review分支中，以便我审核代码，审核通过后我将会合并到main分支中
